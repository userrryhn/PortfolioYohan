Production – Projet BELLETABLE

1. Présentation

Déploiement complet d'une solution de gestion de parc basée sur GLPI. Infrastructure fonctionnelle, sécurisée et administrable.

2. Infrastructure mise en place

  • VM sous Proxmox
  • Debian 12
  • Configuration réseau (IP statique)
  • Apache2
  • MariaDB
  • PHP et modules
  • GLPI

3. Configuration BDD

Base dédiée GLPI. Utilisateur SQL sécurisé. mysql_secure_installation.

4. Paramétrage GLPI

  • Entités
  • Profils utilisateurs
  • Droits d'accès
  • Système de tickets
  • Catégories d'incidents

5. Intégration LDAP

Auth centralisée via LDAP. Config DN, serveur LDAP, filtres. Correction erreurs de bind.

6. Gestion du parc

  • Postes de travail
  • Serveurs
  • Équipements réseau
  • Consommables
  • Attribution aux utilisateurs

7. Sécurisation et sauvegarde

Dump MariaDB via mysqldump. Sauvegarde glpi_backup.sql. Vérification intégrité.

8. Résultat final

Environnement fonctionnel. Connexion LDAP, tickets, consultation du parc opérationnels.

9. Compétences mobilisées

  • Administration Linux
  • Virtualisation
  • Gestion BDD
  • Configuration réseau
  • Droits utilisateurs
  • Diagnostic d'incidents
