Production – Projet BELLETABLE

1. Présentation de la production

La production réalisée dans le cadre du projet BELLETABLE consiste au déploiement complet d’une solution de gestion de parc informatique basée sur GLPI. L’objectif était de fournir une infrastructure fonctionnelle, sécurisée et administrable.

2. Infrastructure mise en place

  • Création d’une machine virtuelle sous Proxmox
  • Installation de Debian 12
  • Configuration réseau (IP statique)
  • Installation Apache2
  • Installation MariaDB
  • Installation PHP et modules nécessaires
  • Déploiement et configuration de GLPI

3. Configuration base de données

Création d’une base de données dédiée à GLPI. Configuration d’un utilisateur SQL sécurisé avec droits restreints. Sécurisation via mysql_secure_installation.

4. Paramétrage fonctionnel dans GLPI

  • Création des entités
  • Création des profils utilisateurs
  • Paramétrage des droits d’accès
  • Configuration du système de tickets
  • Ajout des catégories d’incidents

5. Intégration LDAP

Mise en place d’une authentification centralisée via LDAP. Configuration du DN, du serveur LDAP et des filtres utilisateurs. Tests de connexion et correction d’erreurs de bind.

6. Gestion du parc

  • Ajout des postes de travail
  • Ajout des serveurs
  • Ajout des équipements réseau
  • Gestion des consommables
  • Attribution des équipements aux utilisateurs

7. Sécurisation et sauvegarde

Réalisation d’un dump de la base MariaDB via mysqldump. Sauvegarde du fichier glpi_backup.sql. Vérification de l’intégrité des données.

8. Résultat final

L’environnement est entièrement fonctionnel. Les utilisateurs peuvent se connecter via LDAP, ouvrir des tickets et consulter le parc informatique. La solution répond aux besoins de gestion et de suivi.

9. Compétences mobilisées

  • Administration système Linux
  • Virtualisation
  • Gestion base de données
  • Configuration réseau
  • Gestion des droits utilisateurs
  • Diagnostic et résolution d’incidents
