Situation Professionnelle – Projet BELLETABLE

1. Contexte

Dans le cadre de ma formation en BTS SIO option SISR, j'ai réalisé le projet BELLETABLE. L'objectif était de concevoir et déployer une solution complète de gestion de parc informatique.

Le projet a été réalisé dans un environnement virtualisé via Proxmox. Une VM Debian 12 a été déployée pour héberger GLPI.

Cette situation a été réalisée avec Danis GOUEDART.

2. Architecture technique

  • Hyperviseur : Proxmox VE
  • Système : Debian 12
  • Serveur Web : Apache2
  • BDD : MariaDB
  • Langage : PHP
  • Application : GLPI
  • Auth : LDAP

3. Installation et configuration

  • Installation Apache, MariaDB, PHP
  • Création BDD sécurisée + droits SQL
  • Déploiement GLPI dans /var/www/html

4. Configuration LDAP

Authentification centralisée via LDAP. Correction d'un problème de bind après analyse des paramètres DN.

5. Gestion des profils

  • Administrateur
  • Technicien
  • Utilisateur
  • Observateur

6. Résultats

  • GLPI opérationnel
  • LDAP fonctionnel
  • Système de tickets validé
  • Gestion complète du parc

7. Analyse personnelle

Ce projet m'a permis de consolider mes compétences en administration système, virtualisation et gestion d'infrastructure.
